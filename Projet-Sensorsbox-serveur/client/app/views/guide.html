Get Started

Sensorsbox-client

The recommended way of using SensorsBox is with the javascript library sensorsbox-client.

Installation

Insert the library in your html, and create an instance of the "SensorsBox" Constructor. 

<script src="sensorsbox-client.js"></script>
<script>
  var sbConnection = new SensorsBox.connection();
</script>
This initiates a websocket connection to the server.

Watch a whole box

If you wish to listen to all the sensors of a specific box, you must:

  • Get the box ID on the site
  • call the method "watchBox"
  • listen to "measure" events
<script src="sensorsbox-client.js"></script>
<script>
  var sbConnection = new SensorsBox.connection();
  sbConnection.on('connect', function(){
    sbConnection.watchBox(YOUR_BOX_ID);
  });
  sbConnection.on('measure', function(body){
    console.log(body.data);
  });
</script>

Raw API

The SensorsBox API is built on the Socket.IO library.
To initiate a websocket connection, you must therefore follow the socket.io protocol:

  • Initiate a GET request to 'http://sensorsbox.com/socket.io'
  • Get the first parameter of the response: this is your Socket.IO session ID
  • Initiate a Websocket connection to 'ws://sensorsbox.com/socket.io/1/websocket/YOUR_SESSION_ID'
More information about the protocol can be found on the Socket.IO specification page.
